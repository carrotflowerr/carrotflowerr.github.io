Tools I cannot live without

This isn't going to be a text editor, distro, multiplexer list. You know about tmux and neovim. I wanna put you on to some cool stuff.
  • HTTPrack: can download and mirror sites with faithful js and media rendering. All the have been mirrored and automatically updated by me.
  • Speaking of automation, next we have cron. Cron is a gnu util that can run commands over intervals of time. Using cron is a pain in the ass, but when you learn it it's worth it. There are many bloated electron "gui's" that are meant to simplify cron. But don't.
